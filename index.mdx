## What is OpenClaw Code?

OpenClaw Code (OCCode) is a desktop app that makes it easy to set up and manage your [OpenClaw](https://openclaw.ai) AI agent. No terminal, no config files, no technical knowledge required.

Download it, sign in, and your agent is ready to go.

## How It Works

OCCode connects you to your OpenClaw AI agent through a simple flow:

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   OCCode App     │────▶│   occ.mba.sh    │────▶│   AI Agent      │
│   (Your Desktop) │     │   (Cloud)        │     │   (OpenClaw)    │
└─────────────────┘     └─────────────────┘     └─────────────────┘
    Setup & Manage          Auth + Billing         Your AI Agent
```

| Component | What it does |
|-----------|-------------|
| **OCCode App** | Desktop app for Windows & macOS. Guides you through setting up your agent and managing it day-to-day. |
| **occ.mba.sh** | Cloud service that handles your account, credits, and connects you to your agent. |
| **OpenClaw Agent** | Your personal AI agent, powered by state-of-the-art language models. |

## Key Features

- **No terminal needed** — everything is done through a friendly interface
- **$5 free credits** on signup — enough to get started immediately
- **Simple setup wizard** — get your agent running in minutes
- **Pay-as-you-go** — top up $5 / $10 / $20 / $50 when you need more
- **Open source** — the app and backend are both on GitHub
